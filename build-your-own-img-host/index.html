安装 PicGo 插件

安装完 PicGo 后建议安装github-plus插件

这里的 customUrl 需要配置成JSD地址, 用于加速 github 的文件

什么是 jsd:

jsDelivr - A free, fast, and reliable CDN for open source

jsd 配置格式:

https://cdn.jsdelivr.net/gh/git账号/仓库名
